# Tuya

Tuya Smart is a global leading AI cloud platform service provider (NYSE: TUYA; HKEX: 2391) enabling IoT device manufacturers, solution providers, and app developers to build smart home and industrial IoT applications. The platform provides APIs for device management, smart home automation, scene linkage, data analytics, and industry-specific integrations.

**Type:** Company  
**Modified:** 2026-05-03

## APIs

### Tuya Device Management API
Query, control, and manage IoT devices registered to a Tuya cloud project. Capabilities include device information retrieval, device control commands, factory reset, sub-device listing, operation log queries, and user-device association management.

**Human URL:** [https://developer.tuya.com/en/docs/cloud/device-management](https://developer.tuya.com/en/docs/cloud/device-management?id=K9g6rfntdz78a)

### Tuya Smart Home API
Manage smart home deployments including home and room management, device grouping, scene automation, member management, and data services.

**Human URL:** [https://developer.tuya.com/en/docs/cloud/smart_home_paas](https://developer.tuya.com/en/docs/cloud/smart_home_paas?id=Kakujwbddm7fv)

### Tuya Authorization API
Handle authentication for cloud-to-cloud integrations using HMAC-SHA256 request signing with Access ID and Access Secret credentials.

**Human URL:** [https://developer.tuya.com/en/docs/iot/guide-explanation](https://developer.tuya.com/en/docs/iot/guide-explanation?id=Ke0wpiw0dwxun)

### Tuya Industry API
Enterprise IoT capabilities for industrial device registration, management, status queries, and asset management for B2B deployments.

**Human URL:** [https://developer.tuya.com/en/docs/cloud](https://developer.tuya.com/en/docs/cloud)
